## Passo a passo

1. Acesse o **perfil da instituição**.
2. Clique na aba **“Membros”**.
3. Selecione o botão **“Gerenciar membros”**.
4. Na tela de gerenciamento, utilize o campo de busca para pesquisar o nome do usuário que deseja adicionar.
5. Após localizar o usuário, clique em **“Adicionar”** para enviar o convite.

> O membro será vinculado à instituição assim que aceitar o convite enviado.

## Funcionalidades planejadas para facilitar a adição de membros

Nas próximas fases de desenvolvimento do aplicativo, serão disponibilizadas duas novas opções para tornar o processo de adição de membros ainda mais ágil:

### **Código de Compartilhamento**

- A instituição poderá gerar um link de convite.
- Qualquer pessoa que acessar esse link será automaticamente vinculada à instituição.

### **Escanear Código QR**

- O responsável ou moderador da instituição poderá escanear o QR Code gerado no perfil do usuário (aluno).
- Para isso, o usuário deverá acessar seu perfil e clicar no botão **“Compartilhar”** para exibir o código.
