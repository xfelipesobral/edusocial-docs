## Introdução

Este tutorial apresenta o passo a passo para realizar o cadastro de uma instituição no **EduSocial**. O processo é simples, rápido e facilita o gerenciamento das informações institucionais na plataforma.

## Acessando a opção de cadastro

1. No aplicativo, acesse o **perfil do usuário logado** (ícone disponível no menu principal inferior).
2. Clique na **engrenagem de configurações** no canto superior da tela de perfil.
3. Localize a seção: **“Responsável por uma instituição?”**
4. Clique no botão amarelo correspondente para iniciar o cadastro da instituição.

## Preenchendo os dados da instituição

Você precisará informar os seguintes dados para o cadastro:

- **CNPJ**
- **Nome da instituição**
- **Nome de exibição**
- **E-mail institucional**
- **CEP**
- **Cidade**
- **Estado**
- **Rua**
- **Número**
- **Bairro**
- **Complemento** (opcional)

> Ao informar o **CNPJ**, o aplicativo realizará uma consulta automática na Receita Federal, preenchendo de forma prévia os campos disponíveis. Isso agiliza e facilita o processo de cadastro.

## Finalizando o cadastro

Após preencher todas as informações, clique no botão: **“Enviar para análise”**
