## 📋 Introdução

Este tutorial apresenta o passo a passo para a criação de um usuário no **EduSocial**, detalhando as informações necessárias para o cadastro e destacando as permissões obrigatórias para o acesso à plataforma.

## 🧑‍💼 Como criar um usuário

1. Na **tela de login** ou na **tela de apresentação**, localize e clique no botão: **“Não tem uma conta? Criar agora”**
2. Preencha os dados solicitados no formulário de cadastro:
   - **Nome completo**
   - **Apelido**
   - **E-mail principal**
   - **Data de nascimento**

## ✅ Permissões e consentimentos

Após preencher os dados pessoais, será necessário marcar as opções de consentimento. As opções disponíveis são:

- Receber e-mails
- Receber notificações
- **Aceitar os Termos de Uso** _(obrigatório)_
- **Aceitar a Política de Privacidade** _(obrigatório)_

> ⚠️ **Atenção:** A aceitação dos **Termos de Uso** e da **Política de Privacidade** é obrigatória para concluir o cadastro.

## 🔒 Definindo a senha

- Crie uma senha com, no mínimo, **8 dígitos**.
- Após inserir a senha, clique no botão: **“Completar cadastro”**

## 🎉 Conclusão

Pronto\! O usuário estará cadastrado e poderá acessar normalmente todas as funcionalidades disponíveis no aplicativo **EduSocial**.
