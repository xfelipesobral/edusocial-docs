## Instalação do Aplicativo

Antes de iniciar os testes, é necessário realizar a instalação do aplicativo em seu dispositivo móvel:

- **Android:**\
  Baixe o arquivo `.apk` através deste [link de download](#).
- **iOS:**\
  Para testar no sistema iOS, é necessário utilizar um dispositivo MacOS para compilar o projeto. Siga o passo a passo disponível [neste link](https://github.com/EduSocialApp/mobile).

## Configuração Inicial para Testes

Para garantir a validação completa das funcionalidades do primeiro protótipo, é necessário criar no mínimo **3 usuários** com os seguintes papéis:

- **Aluno**
- **Responsável pelo aluno**
- **Responsável pela instituição**

Essa configuração permite testar os principais fluxos de interação previstos para cada perfil de usuário.

## Funcionalidades Básicas a Serem Testadas

Durante esta etapa, as seguintes funcionalidades devem ser avaliadas:

- [Cadastro e autenticação de usuários](https://edusocial.mintlify.app/essenciais/criando-usuario)
- Envio e gerenciamento de **comunicados institucionais**
- Recebimento de **notificações** sobre eventos, atividades escolares e alertas emergenciais
- Acesso ao **calendário** com datas e eventos relevantes da instituição
